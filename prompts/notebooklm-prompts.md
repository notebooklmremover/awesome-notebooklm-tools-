# NotebookLM Prompt Collection

A curated collection of effective prompts for Google NotebookLM.

> **Full documentation and tools:** [notebooklmremover.org](https://notebooklmremover.org)

## Slide Generation

### Professional Presentation
```
Create a professional 10-slide presentation about [topic].
Use clear headings, bullet points, and include data visualizations where appropriate.
Target audience: [audience]. Tone: [formal/casual/academic].
```

### Academic Lecture Slides
```
Generate lecture slides covering [topic] for a university course.
Include definitions, key concepts, examples, and discussion questions.
Format: title slide + 12 content slides + summary slide.
```

### Business Pitch Deck
```
Create a startup pitch deck for [company/product].
Include: problem, solution, market size, business model, traction, team, ask.
Keep it concise — max 12 slides, minimal text per slide.
```

## Audio Overview (Podcast)

### Deep Dive Discussion
```
Generate a deep dive podcast discussion about [topic].
Two hosts should explore different angles, ask each other questions,
and provide practical takeaways. Duration: ~10 minutes.
```

### Study Guide Audio
```
Create a study guide podcast for [subject/exam].
Cover the key concepts, common exam questions, and memory tricks.
Make it conversational and easy to follow while walking or commuting.
```

### News Summary
```
Summarize the key findings from these documents as a news-style podcast.
Lead with the most important takeaway, then explore supporting details.
Keep it under 5 minutes.
```

## Infographic

### Data Summary
```
Create an infographic summarizing the key data points from these documents.
Use a vertical layout with icons, charts, and statistics.
Color scheme: [blue/green/neutral]. Style: [modern/minimal/corporate].
```

### Process Flow
```
Design an infographic showing the step-by-step process of [topic].
Use numbered steps with icons. Include time estimates and tips for each step.
```

### Comparison Chart
```
Create a comparison infographic between [A] and [B].
Include: features, pricing, pros/cons, and a recommendation.
Use a side-by-side layout with contrasting colors.
```

## Video Generation

### Explainer Video
```
Create a short explainer video about [topic].
Start with the problem, then show the solution.
Use simple language suitable for a general audience.
Include text overlays for key points.
```

### Tutorial
```
Generate a step-by-step tutorial video for [task].
Show each step clearly with on-screen text.
Pace: slow enough for beginners to follow along.
```

## Tips for Better Results

1. **Be specific** — Include audience, tone, and format requirements
2. **Upload quality sources** — Better input documents = better output
3. **Iterate** — Generate, review, refine your prompt, regenerate
4. **Use constraints** — "Max 10 slides", "Under 5 minutes", "3 key points"
5. **Specify style** — "Academic", "Casual", "Corporate", "Minimal"
