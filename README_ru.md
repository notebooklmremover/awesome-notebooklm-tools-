## Зачем это нужно

Google NotebookLM — отличный инструмент для исследований на основе ИИ, но бесплатная версия добавляет водяные знаки ко всем экспортируемым материалам. Официальная опция удаления (NotebookLM Ultra) стоит **$250/месяц**.

Этот репозиторий собирает **бесплатные, открытые и конфиденциальные** инструменты.
