# Contributing

Thanks for your interest in contributing to Awesome NotebookLM Tools!

## How to Contribute

### Add a Tool or Resource

1. Fork this repository
2. Add your tool/resource to the appropriate section in `README.md`
3. Follow this format:
   ```markdown
   | [Tool Name](https://example.com) | Supported formats | Privacy model | Free? |
   ```
4. Submit a Pull Request

### Add a Prompt

1. Add your prompt to `prompts/notebooklm-prompts.md`
2. Include a brief description of what it does
3. Submit a Pull Request

### Report an Issue

- Use the [issue templates](.github/ISSUE_TEMPLATE/) to report broken links, suggest new tools, or request features

## Guidelines

- Only add tools that are **free** or have a **generous free tier**
- Tools must be relevant to NotebookLM, Gemini, or AI content processing
- Prefer **privacy-first** tools (browser-local > server-side)
- No affiliate links
- Keep descriptions concise and factual

## Code of Conduct

Be respectful and constructive. We're all here to help each other get the most out of NotebookLM.
